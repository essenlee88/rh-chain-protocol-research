📌 核心发现：Base 链 Clanker 一家独大（$1.19B 7d vol，占 Base 全部 launchpad 量的 ~90%+），形成强赢家通吃格局。 RH chain 则处于多协议竞争早期阶段（Pons/FLAP/Trench/Noxa 各有 10-40M 量），更像 Solana pump.fun 爆发前的状态。 对 MemeRush 的启示：在 Base 只需核心接入 Clanker 即可覆盖绝大部分流量； 在 RH chain 需同时覆盖多个头部协议（Noxa/FLAP/Pons/Trench/Doppler），格局尚未固化。
⚠️ 数据注意：Clanker 头部 token "OpenAI" $112M 7d vol，仅 13 holders，高度集中； 同期 CZ token $88.9M。这两个 token 贡献了 Clanker 总量的约 17%，可能含 wash trade 成分。 剔除头部 2 token 后 Clanker 净量约 $970M，仍是绝对主导。
